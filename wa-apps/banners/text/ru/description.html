Удобные анимированные баннеры со статистикой кликов

Приложение позволяет с легкостью создавать и добавлять на сайт, блог или магазин анимированные баннеры.

Для добавление баннера в шаблон используется блок вида {$wa->banners->banner("0")} без необходимости подключать какие-либо другие библиотеки, плагины или модули.

Для каждого из баннеров можно загрузить неограниченное количество изображений и выбрать одну из 28 анимации перехода между изображениями.

Основные функции:

  • удобное управление всеми вашими баннерами;
  • быстрое и удобное добавление баннера на сайт без необходимости подключения дополнительных модулей;
  • статистика по переходам для каждого из изобажений;
  • легкое управление всеми параметрами баннера: тип анимации, скорость перехода, время показа каждого изображения, стиль для кнопок переключения слайдов;
  • возможность автоматически обрабатывать изображения под размер баннера;
  • быстрое добавление изображений для баннера;
  • сортировка изображений перетаскиванием мышкой;
  • возможность временно отключить любое из изображений;
  • отображение на одной странице любого количества баннеров.
